## Build Process
1. Clone Repo: `git clone https://github.com/seng350/seng350f19-project-2-1.git`
2. In root directory: 
   - `npm install`
   - Compile: `tsc`
   - Start App: `npm start`
   - Open `http://localhost:3000` in browser
3. To run tests:
   - Unit Tests: `npm test`
   - Unit Tests and Code Coverage: `jest --coverage`
